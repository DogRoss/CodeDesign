when solution is open:
at the top of the screen, middle left is a dropdown that says: debug or release, make sure this is release.
at the top, towards more the left, a option called 'build' is there, click this and click 'build solution'

the file will be in a x64 folder under a 'release' folder
click the exe present and it will run.

the program will present a list of options, the options can be selected by single letter inputs of the first word.
example:
(i)nsert // typing 'i' and pressing enter will call the insert function and prompt you with instructions to input a value.
other options have different letters with other types of instructions

if you want to search or delete you need the hash key which can be found by printing table
