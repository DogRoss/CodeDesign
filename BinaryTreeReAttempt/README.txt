when solution is open:
at the top of the screen, middle left is a dropdown that says: debug or release, make sure this is release.
at the top, towards more the left, a option called 'build' is there, click this and click 'build solution'

the file will be in a x64 folder under a 'release' folder
click the exe present and it will run.

the BT has a UI input that presents a box to input a int value
you can press insert to insert the inputed value or delete to delete it(if it exists)
